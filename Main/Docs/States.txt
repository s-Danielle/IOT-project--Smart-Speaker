
STATE 1 — IDLE (NO CHIP)
User Action	Result	Feedback
Scan chip	Load chip → IDLE (Chip Loaded)	Chip-loaded beep
Play/Pause	No effect	Error beep
Record (hold 3s)	No effect	Error beep
Stop (short)	No effect	Error beep
Stop (long 5s)	No effect	Error beep

STATE 2 — IDLE (CHIP LOADED)
User Action	Result	Feedback
Scan same chip	No effect	Soft no-action click
Scan different chip	Replace chip	Chip-loaded beep
Play/Pause	Start playback → PLAYING	Play chime
Record (hold 3s → release)	Start recording → RECORDING	Record-start beep
Stop (short)	Clear chip → IDLE (No Chip)	Stop tone
Stop (long 5s)	Clear chip → IDLE (No Chip)	Reset tone (longer)

STATE 3 — PLAYING
User Action	Result	Feedback
Play/Pause	Pause → PAUSED	Pause click
Record (hold 3s → release)	Start recording → RECORDING (music pauses)	Record-start beep
Stop (short)	Stop → IDLE (Chip Loaded)	Stop tone
Stop (long 5s)	Stop + clear chip → IDLE (No Chip)	Reset tone
Scan same chip	No effect	Soft no-action click
Scan different chip	Replace chip → IDLE (Chip Loaded)	Chip-loaded beep

STATE 4 — PAUSED
User Action	Result	Feedback
Play/Pause	Resume → PLAYING	Play chime
Record (hold 3s → release)	Start recording → RECORDING	Record-start beep
Stop (short)	Stop → IDLE (Chip Loaded)	Stop tone
Stop (long 5s)	Clear chip → IDLE (No Chip)	Reset tone
Scan same chip	No effect	Soft no-action click
Scan different chip	Replace chip → IDLE (Chip Loaded)	Chip-loaded beep

STATE 5 — RECORDING
User Action	Result	Feedback
Record (short press)	Save recording → return to PAUSED or IDLE (Chip Loaded)	Record-saved chime
Stop (short)	Cancel (no save) → previous state	Cancel tone
Stop (long 5s)	Cancel + clear chip → IDLE (No Chip)	Reset tone
Scan any chip	Ignored	Blocked beep
Play/Pause	Ignored	Blocked beep
